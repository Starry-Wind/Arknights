## 常见问题

- 脚本停止在主线任务，请手动把主线任务清掉
- 如果连接错误，默认为夜神模拟器，其他模拟器请自行改文件中的adb地址
- 如果脚本不运行或一直出错，请确保你的模拟器分辨率为1920*1080 （平板模式）
